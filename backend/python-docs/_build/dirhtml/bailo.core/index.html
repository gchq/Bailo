bailo.core package

The Bailo core package contains suppport with one to one endpoints with Bailo. It is recommended to use the helper package for most use

class bailo.core.agent.Agent[source]

Bases: object

Base API Agent for talking with Bailo.

Wraps each request in an exception handler that maps API errors to Python Bailo errors, among status codes less than 400. Defaults request timeout to 5 seconds.
