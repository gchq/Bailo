# Scanning Files

## Using Scanners

When uploading files to Bailo, your files will be automatically scanned (assuming scanners are configured and
initialised).

Bailo is configured, by default, to use the scanners: ClamAV and ModelScan (ArtefactScan), see
[Bailo - Artefact Scanners](/docs/administration/microservices/artefact-scanners)

For scan results:

- Go to your model and click on File Management Tab
- Click the Scan chip, towards the right of your file
- View scan results, note the two scanners' results are listed separately

  <Image alt='view file scanning chip' src={chipView} />

## Rescan File

If you want to rescan, click the 3 dots to the right of the scan results chip and select the Rescan option.
